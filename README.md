# moleculate
Утилита для создания структуры папок и файлов на основе методологий БЭМ и атомарного дизайна

**Установка**

> npm i -g moleculate

## Основные команды

### Создание атома

> moleculate new atom

или

> moleculate n a

*Name:*       — название атома

*Mutates:*    — мутации атома через пробел (модификаторы в БЭМ)

*Tag:*        — html-тег атома

*Attributes:* — атрибуты тега в формате <attr>|<value>
